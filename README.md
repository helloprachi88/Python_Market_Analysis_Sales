# Market Analysis Sales using Python

## Project Overview

This project focuses on analyzing sales data using Python to uncover business insights and identify sales patterns. The objective is to perform data cleaning, exploratory data analysis (EDA), and visualization to understand product performance, revenue trends, and customer purchasing behavior.

---

## Objective

The goal of this project is to:

* Analyze historical sales data
* Identify top-performing products and categories
* Study monthly and seasonal sales trends
* Understand customer purchasing patterns
* Generate insights for business decision-making

---

## Dataset

The project uses a CSV sales dataset containing information such as:

2. Cust_name	             6. Age	                    10. Product_Category
3. Product_ID	             7. Marital_Status          11. Orders
4. Gender	                 8. State	Zone	            12. Amount

## Project Workflow

### 1. Data Cleaning

* Removed missing/null values
* Checked duplicate records
* Standardized column formats

### 2. Exploratory Data Analysis (EDA)

* Total sales calculation
* Monthly sales trend analysis
* Product/category-wise sales analysis
* Region-wise sales performance

### 3. Data Visualization

Created charts to visualize:

* Monthly revenue trends
* Best-selling products
* Sales by category
* Regional performance comparison
